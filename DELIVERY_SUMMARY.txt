STEP 1: Generate LaTeX PDF (2 minutes)
───────────────────────────────────────────────────────────────────────────────
Windows:
  1. Download MiKTeX from https://miktex.org/download
  2. Install MiKTeX
  3. Open command prompt in file directory
  4. Run: pdflatex TECHNICAL_REPORT_UVILLAGE_INFRACTIONS.tex

macOS:
  brew install --cask mactex
  pdflatex TECHNICAL_REPORT_UVILLAGE_INFRACTIONS.tex

Linux:
  sudo apt-get install texlive-full
  pdflatex TECHNICAL_REPORT_UVILLAGE_INFRACTIONS.tex

STEP 2: Generate PlantUML Diagrams (1-5 minutes)
───────────────────────────────────────────────────────────────────────────────
Method A (Online - Easiest):
  1. Go to https://www.plantuml.com/plantuml/uml/
  2. Copy from UML_CLASS_DIAGRAM.puml
  3. Paste into text box
  4. Click "Download as PNG"
  5. Repeat for other 2 diagram files

Method B (Command Line):
  brew install plantuml  # macOS
  apt-get install plantuml  # Linux
  
  plantuml -Tpng UML_CLASS_DIAGRAM.puml
  plantuml -Tpng UML_SEQUENCE_DIAGRAM.puml
  plantuml -Tpng UML_ER_DIAGRAM.puml

Method C (VS Code):
  1. Install PlantUML extension by jebbs
  2. Open UML_CLASS_DIAGRAM.puml
  3. Press Alt+D to preview
  4. Right-click → Export Diagram as PNG

STEP 3: Optional - Add Diagrams to Report
───────────────────────────────────────────────────────────────────────────────
1. Place PNG files in same directory as TEX file
2. Add sections after Chapter 3:
   
   \section{Component Diagram}
   \begin{figure}[H]
   \centering
   \includegraphics[width=0.95\textwidth]{UML_CLASS_DIAGRAM.png}
   \caption{Complete Class Diagram}
   \end{figure}

3. Recompile LaTeX
